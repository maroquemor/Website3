# Portafolio Profesional - Ingeniero Mecánico

## Descripción
Portafolio profesional que incluye:
- Hoja de vida detallada
- Carta de presentación
- Proyectos destacados
- Artículos científicos publicados

## Características
- Diseño responsive y moderno
- Navegación entre secciones
- Funcionalidad de impresión y descarga
- Integración de artículos académicos
- Estilo profesional para SAE/ASME

## Estructura de Archivos
- `index.html` - Página principal
- `styles.css` - Estilos principales
- `script.js` - Funcionalidad JavaScript
- `Articulo1_OBD.html` - Artículo científico sobre análisis OBD
- `README.md` - Documentación

## Tecnologías Utilizadas
- HTML5, CSS3, JavaScript
- Font Awesome para iconos
- Diseño responsive con CSS Grid y Flexbox
