## Proyectos Incluidos

### Proyecto 1: Auditoría Integral del Sistema de Aire Comprimido
Evaluación completa del sistema de aire comprimido para optimizar eficiencia energética y calidad del aire según normas ISO.

**Características:**
- Evaluación según normas ISO 11011, 8573, 55000, 14224
- Identificación de ahorros potenciales de $185,400 USD anuales
- Plan de remediación con ROI de 1.8 años

### Proyecto 2: Implementación RCM - Análisis Económico para Planta de Producción
Implementación de Mantenimiento Centrado en la Confiabilidad (RCM) con análisis económico detallado.

**Características:**
- Análisis según normas SAE JA1011, SAE JA1012, ISO 14224
- ROI del 135-220%, período de recuperación de 5-9 meses
- Ahorros anuales de $330,000 - $540,000 USD
- VPN a 5 años de $850,000 - $1,450,000 USD
