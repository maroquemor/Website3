Análisis Multivariado Avanzado de Datos OBD para Diagnóstico Predictivo de Vehículos

Publicación: SAE/ASME 2026 | DOI: 10.4271/2026-01-XXXX

Este artículo presenta un marco integral de análisis multivariado para datos de Diagnóstico a Bordo (OBD), integrando siete técnicas estadísticas avanzadas para habilitar diagnósticos predictivos de vehículos. El análisis revela que la carga del motor explica el 67.6% de la varianza multivariada (Λ de Wilks = 0.324, p < 0.001).
